== Description ==
DGAshu Cron Job Mail Setup For User Role is basically send mail to the User on the basis of User Role. People can set each and every setting for their mail. We included editor but oly given right to send plain mail.

An overview of the features:

*   Set the **Cron Time manual**, so that Mail send after same interval of time. (**Minimum time to be set is 60sec as per the Wordpress Cron Standard.**)
*   Set the **Subject of the Mail** for the User.
*   Set the **Plain/text message** in text area. This message will be send to the selected role user. (**We are soon going to make it HTML/text Area**)
*   Select the **User Role** for the mail to be sent. For example if we select **subscriber** then mail will sent to all the subscriber of the website.

This plugin is under active development. Any feature requests are welcome!

== Installation ==

1. Upload the `custom-related-posts` directory (directory included) to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
